================================================================================
                             底栏导航（Tabs 原生）
================================================================================

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│                              主内容区                                    │
│                        (TabContent 嵌入)                                 │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│        [ 人物 ]                    [ 故事 ]                             │
│           ●                           ○                                 │
│      (选中·品牌色)              (未选中·灰色)                            │
│      下划线 3vp                                                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

  Tab 切换行为：
    · 点击 Tab 或手指横向滑动切换
    · 切换后保持滚动位置和状态
    · PersonPage / StoryPage 直接嵌入，非 router 跳转
    · 详情页 (PersonDetailPage / StoryDetailPage) 保持 router.pushUrl

================================================================================
                             关键页面跳转矩阵
================================================================================

┌──────────────────┬──────────────────┬──────────────────┬──────────────────┐
│  起点 \ 终点      │ PersonPage      │ StoryPage       │ DetailPage      │
├──────────────────┼──────────────────┼──────────────────┼──────────────────┤
│ PersonPage       │ —               │ Tab 切换         │ 点击卡片行       │
│                  │                  │ (保持状态)       │ pushUrl          │
├──────────────────┼──────────────────┼──────────────────┼──────────────────┤
│ StoryPage        │ Tab 切换         │ —               │ 点击卡片行       │
│                  │ (保持状态)       │                  │ pushUrl          │
├──────────────────┼──────────────────┼──────────────────┼──────────────────┤
│ PersonDetailPage │ router.back()    │ —               │ → StoryDetail   │
│                  │                  │                  │ (点故事封面)     │
├──────────────────┼──────────────────┼──────────────────┼──────────────────┤
│ StoryDetailPage  │ —               │ router.back()    │ → PersonDetail  │
│                  │                  │                  │ (点人物头像)     │
└──────────────────┴──────────────────┴──────────────────┴──────────────────┘
