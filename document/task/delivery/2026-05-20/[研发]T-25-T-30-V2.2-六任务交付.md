## 设计决策

| 决策 | 说明 |
|------|------|
| T-25 移除仅删关联 | `DELETE FROM photo_owner_link` + DECR `reference_count`，不删 `photo_ref` 行（其他 Story/Person 可能仍引用） |
| T-26 NULLS LAST | `pr.taken_at_cache` 从未写入时排序等效于 `pol.linked_at DESC`，未来写入后自动生效 |
| T-27 GC 方式 | DEBUG 手动按钮触发（用户决策），不做自动触发 |
| T-27 GC 删除策略 | 硬删除孤立 `photo_ref`（LEFT JOIN WHERE pol.id IS NULL），无保留价值 |
| T-28 名称缓存 | `aboutToAppear` 中查名称存入 `@State ownerName`，构建时直接引用 |
| T-29 对话框 | 阻断后不清除候选，用户可减少勾选后重试 |
| DomainRules 类型修复 | 发现 `Rules.ets` 裸对象字面量不符合 `arkts-no-untyped-obj-literals`，添加显式 `DomainRulesType` 接口 |
