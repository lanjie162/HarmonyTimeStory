## 6.1 · Agent / CI 全程代跑（构建 → 安装 → `aa test` → 可选 hilog）（2026-05-10+）

适用于 **Hypium 用例扩充后**（含 DEBUG mock 相册等价 URI、导入 PH-01 双占位、B2 负例与新取消矩阵）；**必须用 debug 签名包**，与 §2、§7 的产物路径一致。

**要点**：`aa test` stdout 常为 **UTF-16**；**禁止**对该流做 `Tee-Object` 到 UTF-8 头再追加（会混编码）。应将 **单次 `hdc shell` 的原始输出整体**写成 UTF-8（见下文 `Out-File -Encoding utf8` 单行重定向），或 MCP/脚本在落盘前先按设备实际编码规整。

### 步骤（顺序固定）

1. **构建**：DevEco MCP `build_project`，`module=entry@default` 与 **`module=entry@ohosTest`**，`build_intent=LogVerification`，`clean` 按需；或使用 IDE **Assemble Hap**。
2. **`hdc list targets`**；若空：`hdc tconn <IP:port>`。
3. **`hdc install`**：`entry-default-signed.hap`、`entry-ohosTest-signed.hap`（路径见 §2）。
4. **`aa test`**：超时须 **≥600000 ms**（用例增多 + 多次全量扫描）。
5. **（可选）hilog**：`hdc hilog -x`，管道 `Select-String` 过滤 `cancelRequest|扫描已取消`，取 **30～50 行**写入 `*-hilog-tail.txt`。

### PowerShell 可复制模板（stdout 单列 UTF-8 文件）

```powershell
$HDC = "C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe"
$Root = "c:\coding\DevEcoStudioProjects\timestore"
$HAP = Join-Path $Root "entry\build\default\outputs\default\entry-default-signed.hap"
$TESTHAP = Join-Path $Root "entry\build\default\outputs\ohosTest\entry-ohosTest-signed.hap"
$Out = Join-Path $Root "document\evidence\V2\2026-05-10-T9-V2D1-D4-hypium-run26-album-b2-ph01.txt"

& $HDC list targets
& $HDC install $HAP
& $HDC install $TESTHAP

$cmd = 'aa test -b com.lanjie162.timestore -m entry_test -s timeout 600000 -s unittest OpenHarmonyTestRunner'
$header = "# $(Get-Date -Format o)`r`nhdc shell: $cmd`r`n`r`n"
[IO.File]::WriteAllText($Out, $header + (& $HDC shell $cmd *>&1 | Out-String))

# 可选（同一 UTF-8 策略另存 hilog）
# & $HDC hilog -x | Select-String "cancelRequest|扫描已取消" | Select-Object -First 40 | Out-File (Join-Path $Root "document\evidence\V2\2026-05-10-T9-V2D1-D4-hypium-run26-hilog-tail.txt") -Encoding utf8
```

**通过判据**：`$Out` 内 **`OHOS_REPORT_RESULT`**：`Failure: 0`、`Error: 0`，且 **`TestFinished-ResultCode: 0`**。当前工程预期 **`Tests run: 16`**（`ActsAbilityTest` 1 条 + **`V2Regression` 15 条**，含相册 mock / PH-01 / B2 矩阵）。

**已归档取证（与本节 `$Out` 同名）**：`document/evidence/V2/2026-05-10-T9-V2D1-D4-hypium-run26-album-b2-ph01.txt` — **Pass 16/16**（2026-05-11）；主档已与 **`hypium-run28-ime-dismiss.txt`**（`SoftKeyboard.dismissSoftKeyboard` / `stopInputSession` 后主键）同步。历史副本：`…-hypium-run27-dev-nested-scroll.txt`（嵌套滚动优化轮）。
