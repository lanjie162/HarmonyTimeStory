# T-6 · 导入主链最小证据（会话内）

- **关联派发**：`TR-20260509-02#C-3`（计划任务 V2-T3）
- **代码入口**：`entry/src/main/ets/pages/import/ImportPage.ets`
- **上下文入参**：人物详情 → `params: { ctx: 'person', ownerId }`；故事详情 → `ctx: 'story', ownerId`
- **主顺序（UI）**：选图 → 导入准备 → 条件 → 全屏扫描（进度条）→ 候选确认 → 确认写入 → `router.back()`；返回后详情 **`onPageShow`** 触发刷新以复现照片网格。
- **与稿偏差登记**：全屏进度为 **P-2 骨架**（半透明遮罩 + `Progress`），若与 UI v1.2 像素稿不一致，走计划并行裁决条文，由 T-8 收敛。

## 最小复现

1. Shell → 人物 → 新建并进入人物详情（或故事详情）。
2. 点「进入导入向导壳页」。
3. 选图 → 经准备、条件 →「开始扫描」；扫描中途可点「取消扫描」验证中断（与 T-7 共用 `cancelRequest`）。
4. 扫描完成后「确认写入」，返回详情页应看到照片网格更新（含 PH-01 去重 toast 语义，见执行结果表）。
